# BookBurners

BookBurners is a lightweight web app where readers can discover books, join clubs, and discuss titles with other readers.

## Features
- Browse and view book details
- Create and participate in discussions
- User authentication (login / register)
- Simple, responsive UI (HTML, CSS, JS)
- PHP backend with small API endpoints

## Quick start

Requirements:
- PHP 7.4+ (or a local web server)
- Composer (optional, for dependencies)

Run locally (recommended):
1. Edit configuration if needed: see [php/config.php](php/config.php).
2. From the project root run the built-in PHP server:
   ```sh
   php -S localhost:8000 -t .
